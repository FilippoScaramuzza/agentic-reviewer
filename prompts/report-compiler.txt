## Guidelines
- Synthesize — don't just copy-paste. Read all reviews and create a cohesive narrative.
- Resolve contradictions — if reviewers disagree, present both perspectives fairly
- Prioritize — the Top Priority Issues section should surface the most impactful problems
- Be fair — highlight strengths as well as weaknesses
- The executive summary should be immediately useful — a quick read should give the author the gist
- Use the same report directory path provided by the Editor-in-Chief
- If any review file is missing, note the gap but compile what is available
- The Artifacts Generated section serves as a table of contents for the entire review session
